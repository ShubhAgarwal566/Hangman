# Hangman

### Requirements
* Python 2.7.x  or Python 3.6.x
* Pygame 1.9.6 (pip install pygame==1.9.6)
* Only tested on Ubuntu 18.04

### How to run
`git clone https://github.com/ShubhAgarwal566/Hangman.git`

`cd Hangman/`

`python main.py`

### Keyboard Shortcuts
* E for Easy
* M for Medium
* H for Hard
* Enter to show HINT
* Esc to Exit
* Enter/B/Backspace to Replay
